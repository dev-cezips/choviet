## 🧬 비전 (늦깎이연구소 커널 적용)

### 정체성 재정의
```
기존: "베트남인 중고거래 앱"
    ↓
신규: "두 문화를 연결하는 커뮤니티 플랫폼"
```

### 핵심 철학
```
성찰(Reflection) → 믿음(Belief) → 욕망(Desire)
      ↓               ↓              ↓
   기록하다        연결하다        성장하다
      ↓               ↓              ↓
  거래 히스토리    동포 네트워크    커뮤니티 성장
```

### 5대 키워드 적용
| 늦깎이연구소 | Choviet 적용 |
|-------------|-------------|
| 다시 시작 | 한국 정착 새 출발 |
| 성장 | 신뢰 점수 성장 |
| 가족 | 다문화가족 연결 |
| 연결 | 베트남 동포 네트워크 |
| 기회 | 중고거래/구인구직 |

### 차별화 포인트
- "매너온도" → **"신뢰 기록"** (기록의 철학)
- "거래 후기" → **"성장 스토리"** (성찰의 철학)
- "카테고리" → **"삶의 영역"** (연결의 철학)
