# 🚀 Chợ Việt 베타 테스트 운영 전략

> "베트남 사람들이 한국어 스트레스 없이 소통하는 첫 번째 플랫폼"

## 📅 베타 테스트 타임라인

### Phase 1: Soft Launch (Week 1-2)
- **목표**: 10-20명의 핵심 사용자 확보
- **대상**: 친한 베트남 친구들, 신뢰할 수 있는 early adopters
- **지역**: 안산 집중 (베트남인 밀집 지역)

### Phase 2: Controlled Growth (Week 3-4)
- **목표**: 50-100명으로 확장
- **대상**: 친구의 친구, 소규모 커뮤니티
- **지역**: 안산 + 수원 확장

### Phase 3: Open Beta (Week 5-8)
- **목표**: 500+ 활성 사용자
- **대상**: 공개 모집
- **지역**: 수도권 전체

## 🎯 초기 사용자 획득 전략

### 1. **"언니/형" 전략** (Influencer Seeding)
각 지역별로 영향력 있는 베트남 커뮤니티 리더 섭외:
- 안산: 베트남 식당 사장님들
- 수원: 베트남 유학생회 회장
- 서울: 베트남 맘카페 운영자

**접근 메시지**:
```
언니/형, 안녕하세요! 
한국 생활하면서 베트남어로 편하게 소통할 수 있는 앱을 만들었어요.
한국어 전혀 몰라도 사용할 수 있게 만들었는데, 
언니/형이 먼저 써보고 피드백 주시면 안 될까요? 
커피 한잔 사드릴게요! ☕
```

### 2. **"Teencode 챌린지"** (Viral Hook)
- 첫 주 이벤트: "가장 창의적인 Teencode 메시지 공유하기"
- 상품: 베트남 커피 세트, 쌀국수 식사권
- 목적: 자연스러운 입소문 + Teencode 번역 데이터 수집

### 3. **"동네 장터 오픈"** (Practical Value)
- 안산 원곡동 중심으로 "베트남 물건 나눔/판매" 활성화
- 실용적 가치로 daily active 유도
- 첫 거래 성사시 양쪽에 5,000원 쿠폰

### 4. **"SOS 번역"** (Community Help)
- "긴급 번역 요청" 기능으로 즉각적인 도움 제공
- 병원 서류, 학교 가정통신문 등 실생활 번역
- 도움 준 사람에게 포인트/배지 제공

## 📊 핵심 추적 지표 (North Star Metrics)

### 1. **Activation Rate**
- **정의**: 가입 후 72시간 내 첫 게시글 작성 또는 메시지 발송
- **목표**: 60% 이상
- **측정**: `user_signup` → `post_created` OR `message_sent`

### 2. **Weekly Engagement**
- **정의**: 주 3회 이상 방문하는 사용자 비율
- **목표**: 40% 이상
- **측정**: Unique active users with 3+ sessions/week

### 3. **Translation Usage**
- **정의**: 전체 메시지 중 번역 기능 사용 비율
- **목표**: 30% 이상 (한-베 소통 활성화 지표)
- **특별 추적**: Teencode 포함 메시지 비율

### 4. **Trust Score**
- **정의**: (성공 거래 + 긍정 피드백) / (전체 거래 + 신고)
- **목표**: 85% 이상
- **측정**: 거래 완료 + 신고 데이터

## 🎬 사용자 행동 유도 시나리오

### Week 1: "첫 인사"
```
푸시 알림: "안녕하세요! 동네에 사는 베트남 이웃들과 인사해보세요 👋"
→ 동네 피드 진입 유도
→ 간단한 자기소개 게시글 작성 유도
```

### Week 2: "첫 도움"
```
푸시 알림: "병원 서류 번역이 필요하신가요? SOS 번역 요청해보세요!"
→ 번역 기능 사용 유도
→ 커뮤니티 도움 경험
```

### Week 3: "첫 거래"
```
푸시 알림: "이번 주 인기 매물: 베트남 식재료, 아기 용품 🛍️"
→ 마켓플레이스 탐색
→ 첫 거래 시도
```

### Week 4: "커뮤니티 정착"
```
푸시 알림: "이번 주 모임: 안산 베트남 맘들의 차 한잔 ☕"
→ 오프라인 연결
→ 정기 사용자화
```

## 🔍 데이터 수집 및 분석

### 1. **정량적 데이터** (Analytics Events)
- **User Journey**: 가입 → 첫 행동 → 정착 → 이탈
- **Feature Usage**: 어떤 기능을 가장 많이 쓰나?
- **Translation Patterns**: Teencode 사용 패턴
- **Trust Metrics**: 신고율, 거래 성공률

### 2. **정성적 데이터** (User Feedback)
- **주간 사용자 인터뷰**: 5명씩 30분 화상 인터뷰
- **In-app 피드백**: 간단한 이모지 피드백
- **WhatsApp 그룹**: 실시간 피드백 채널

### 3. **A/B 테스트 아이디어**
- 온보딩: 위치 설정 먼저 vs 관심사 선택 먼저
- 번역: 자동 번역 vs 요청시 번역
- 알림: 베트남어 only vs 이중 언어

## 🚨 위험 요소 및 대응 방안

### 1. **콜드 스타트 문제**
- **리스크**: 초기 콘텐츠 부족
- **대응**: 팀이 직접 유용한 콘텐츠 시딩 (맛집, 생활 팁)

### 2. **언어 장벽 역설**
- **리스크**: 한국인 참여시 베트남인들이 불편
- **대응**: "베트남어 우선" 정책 명확히 공지

### 3. **스팸/사기**
- **리스크**: 초기 신뢰도 하락
- **대응**: 24시간 모니터링 + 빠른 대응

### 4. **기술 문제**
- **리스크**: 서버 다운, 버그
- **대응**: 매일 백업 + 핫픽스 준비
