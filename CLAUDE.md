# Choviet 프로젝트 컨텍스트

## 프로젝트 개요
- **이름**: Chợ Việt (쵸비엣)
- **비전**: 두 문화를 연결하는 커뮤니티 플랫폼
- **상위 조직**: 늦깎이연구소
- **웹사이트**: https://choviet.chat

## 🧬 커널 구조 (늦깎이연구소 철학)

```
성찰 → 믿음 → 욕망
  ↓      ↓      ↓
기록 → 연결 → 성장
```

### 5대 키워드 → Choviet 적용
| 키워드 | 적용 |
|--------|------|
| 다시 시작 | 한국 정착 새 출발 |
| 성장 | 신뢰 점수 성장 |
| 가족 | 다문화가족 연결 |
| 연결 | 베트남 동포 네트워크 |
| 기회 | 중고거래/구인구직 |

### 숨겨진 키워드
> "포기 안 함" - 늦게 시작했지만 포기하지 않는 사람들

## 늦깎이연구소 시너지 맵

```
lbl_cafe24 (쇼핑몰)
    │
    ├── "두 문화의 기록" 카테고리 ←──── Choviet 타겟 사용자
    ├── 한글+베트남어 이중언어 굿즈 ←── 상품 연동
    └── 다문화가족 전략 ←────────────── 마케팅 채널
         │
Silentia (알림 철학) ←─────────────────── 스마트 알림 적용
         │
Tiến Lên (게임) ←──────────────────────── 미니게임 연동
         │
lbl-wordpress (브레인) ←───────────────── 콘텐츠 연동
```

- 민들레 홀씨 할아버지
- "기록의 수호자"
- Choviet 적용: 거래 안내, 신뢰 배지, 빈 상태 일러스트

## 우선순위 원칙
> 늦깎이연구소 성장 공식: 콘텐츠 > 사람 > 돈

1. **기록** - 사용자 거래 히스토리 = 신뢰 스토리
2. **연결** - 동포 네트워크 구축
3. **성장** - 커뮤니티 활성화 후 수익화

## 기술 스택
- Backend: Ruby on Rails 8.0
- Database: SQLite (dev) / PostgreSQL (prod)
- Real-time: Redis + ActionCable
- Deploy: Kamal (AWS EC2)
- Domain: choviet.chat

## 주요 명령어
```bash
# 개발
bin/dev

# 배포
kamal deploy

# 콘솔
kamal app exec -i 'bin/rails console'
```
